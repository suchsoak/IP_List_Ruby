# IP_List_Ruby

Shows active IPs that are connected to the WiFi network

This script is under development

# Download 

> Download the requirements.sh

```sh
   git clone https://github.com/suchsoak/IP_List_Ruby.git
```

```sh
  ruby list_ip.rb
```

# Usage

Put the first ip

```sh
  000.000.0.1
```

Then, the final ip

```sh
  000.000.0.255
```
Then the script will be printed

```sh
  IP | PORT
```
